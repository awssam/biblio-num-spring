Comment ajouter un livre ?

Pour ajouter un nouveau livre dans la bibliothèque, veuillez remplir tous les champs obligatoires (marqués d'un *) et les informations supplémentaires si disponibles.

Pour une meilleure expérience utilisateur :

  • Ajoutez une couverture pour améliorer la visibilité
  • Sélectionnez les bonnes catégories pour faciliter la recherche
  • Rédigez un résumé complet pour attirer les lecteurs
